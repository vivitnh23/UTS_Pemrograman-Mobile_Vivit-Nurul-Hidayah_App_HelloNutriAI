# HelloNutri AI

**HelloNutriAI** adalah aplikasi asisten nutrisi personal berbasis AI yang dirancang untuk mendampingi perjalanan gaya hidup sehat kamu. Aplikasi ini tidak hanya mampu melacak asupan kalori secara cerdas, tetapi juga memberikan rekomendasi gizi yang sepenuhnya disesuaikan dengan kebutuhan unik tubuhmu.

---

## 📱 Tampilan UI & Alur Aplikasi

Berikut adalah penjelasan setiap layar pada aplikasi HelloNutri AI:

### 1. Splash Screen & Loading
Layar pembuka yang memberikan kesan *fresh* dan minimalis. Pada tahap ini, aplikasi melakukan inisialisasi awal dan persiapan sistem sebelum masuk ke menu utama.

Layar krusial untuk mengumpulkan data unik pengguna seperti **Nama, Berat Badan, Tinggi Badan, dan Umur**. Data ini berfungsi sebagai dasar bagi AI untuk memberikan saran yang akurat dan personal (Personalized Nutrition).

## Tech Stack
- **Bahasa Pemrograman**: Java / Kotlin (Android Studio)
